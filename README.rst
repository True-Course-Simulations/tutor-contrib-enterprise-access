enterprise-access plugin for Tutor


Installation
************

.. code-block:: bash

    pip install git+https://github.com/True-Course-Simulations/tutor-contrib-enterprise-access

Usage
*****

.. code-block:: bash

    tutor plugins enable enterprise-access
